## Testing the API

1. Navigate to the API package and install dependencies:

```bash
cd api && yarn install
```

2. Run the unit test suite:

```bash
yarn test
```

The API uses Jest with TypeScript via `ts-jest`. Ensure dependencies are installed before running the tests.
